# RAnesthesia (Web PWA)

Public, English-only anesthesia resident education assistant (educational use only).

## Run locally
1) Install Node.js (v18+)
2) In this folder:
   - `npm install`
   - `cp .env.example .env` and put your API key in `.env`
   - `npm run dev`
3) Open http://localhost:3000

## Deploy
- Import to Vercel
- Add Environment Variable: `OPENAI_API_KEY`
- Redeploy
